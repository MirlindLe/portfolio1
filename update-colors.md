# Color Replacement Guide

## Theme Color Mappings:

- `text-white` → `text-foreground`
- `text-gray-300` or `text-gray-400` → `text-muted-foreground`
- `text-teal-400` or `text-teal-300` or `text-cyan-400` → `text-primary`
- `text-purple-400` → `text-accent`
- `bg-white/5` or `bg-white/10` → `bg-card`
- `border-white/10` or `border-white/20` → `border-border`
- `bg-teal-500/20` or `bg-cyan-500/20` → `bg-primary/20`
- `bg-gradient-to-r from-teal-500 to-cyan-500` → `bg-primary`
- `bg-gradient-to-r from-purple-500 to-blue-500` → `bg-accent`
- `text-green-400` → `text-primary`
- `hover:text-teal-400` → `hover:text-primary`
- `hover:text-white` → `hover:text-foreground`
